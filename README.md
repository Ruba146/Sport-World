# Sport World

Sport World is an interactive web application that allows users to explore different sports by searching for a sport name and selecting what information they want to view, such as rules, tournaments, famous players, or suitable age and gender.

This website integrates real-time APIs to fetch and display up-to-date sport information.
---

## Features
- Search for any sport by name  
- Choose what to explore: rules, tournaments, players, or age & gender  
- Real-time sport validation using Wikipedia API  
- Dynamic results with images and descriptions  
- External links to Wikipedia and related resources  
- Clean and modern user interface  
- Responsive layout for different screen sizes  

---

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Wikipedia REST API  
- Google Custom Search API  

---

## How It Works
1. User enters a sport name.  
2. System checks if it is a real sport using Wikipedia.  
3. Basic sport information and image are displayed.  
4. Additional data is fetched from Google based on the selected category.  
5. Results are displayed dynamically without page reload.
